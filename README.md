# Full Spectrum Lab

This organization hosts the public protocol, engine, and commons layers of the Full Spectrum ecosystem.

- `full-spectrum-protocol`: protocol language and RFCs
- `full-spectrum-engine`: executable engine and validation artifacts
- `full-spectrum-enterprise-governance`: enterprise adaptation and deployment patterns
- `full-spectrum-commons`: project-wide public assets, coordination notes, and cross-repo indexes


Full Spectrum Lab builds the protocol layer, engine layer, and public commons layer for human-AI governance.

Full Spectrum Lab 聚焦人机协作的协议层、引擎层与公共层：协议规范、可运行引擎、公共索引与协作留痕。

full-spectrum-lab is the public workspace for the Full Spectrum ecosystem. It hosts protocol specifications, runnable engines, reusable examples, and common project assets. The goal is to turn governance ideas into reproducible engineering artifacts.

full-spectrum-lab 是全频谱体系的公开工作空间，承载协议规范、可运行引擎、可复用样例与公共项目资产。目标是把治理理念转化为可运行、可验证、可接力的工程成果。
