# Public Status Source of Truth

[`public-status.json`](./public-status.json) is the machine-readable public release and maturity record for the organization landing page.

Update it only when a Release changes meaning, a development line ships, a designed case gains evidence, or a formally unsupported capability becomes supported.

The validation workflow checks JSON structure, release URLs and consistency with both organization profile files. Gitee and GitHub may retain different commit histories; this file tracks public product meaning, not hash equality.
